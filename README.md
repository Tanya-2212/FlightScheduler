# FlightScheduler — Flight Schedule Management Table

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Vite will run on port `8080`.
If your environment can’t resolve `localhost`, use `http://127.0.0.1:8080` instead.

## Data

Flight data is loaded once on mount from `public/flights.json` and managed fully in client-side state.
